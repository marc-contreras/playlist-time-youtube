# Playlist Time

Chrome extension that calculates the total duration of YouTube playlists and mixes.

## Features

* Calculates total playlist duration
* Works with YouTube playlists and mixes
* Lightweight and fast
* Built with vanilla JavaScript

---

# Installation (Developer Mode)

Since the extension is not published yet, you need to install it manually in Chrome.

## 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/playlist-time.git
```

Or download the ZIP and extract it.

---

## 2. Open Chrome Extensions

Go to:

```txt
chrome://extensions
```

---

## 3. Enable Developer Mode

Activate the toggle in the top-right corner:

```txt
Developer mode
```

---

## 4. Load the extension

Click:

```txt
Load unpacked
```

Then select the project folder:

```txt
playlist-time/
```

---

## 5. Open YouTube

Go to any YouTube playlist or mix.

You should see the total playlist duration displayed on the page.
